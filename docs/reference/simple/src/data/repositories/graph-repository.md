# `src/data/repositories/graph-repository.ts`

This repository saves and reads Vision graph records.

## `upsertNode`

Saves a newer node version.

## `upsertEvent`

Saves planning-safe event facts by provider identity.

## `replaceEdges`

Replaces one node's graph links together.

## `getEventByProviderIdentity`

Finds one event by its complete provider identity.
